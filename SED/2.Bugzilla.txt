**AIM**
To study bug reporting using Bugzilla.

---

**THEORY**
A bug is an error or defect in a software system that causes incorrect or unexpected results.
Bugzilla is an open-source bug tracking tool used to report, track, and manage bugs.
It helps in identifying issues, assigning them to developers, and monitoring their status.

Bug Lifecycle:
New → Assigned → Open → Fixed → Closed

Bug Fields:

* Product
* Component
* Summary
* Description
* Severity
* Priority

---

---

**STEPS TO REPORT BUG IN BUGZILLA**

1. Login to Bugzilla
2. Click on **New Bug**
3. Select Product and Component
4. Enter Summary and Description
5. Set Severity and Priority
6. Submit bug report
